# MathVerse Applications Directory

This directory contains all frontend applications for the MathVerse platform.

## Applications

- **backend/**: FastAPI REST API service
- **web/**: Next.js web application
- **mobile/**: Flutter mobile application
- **creator-portal/**: Teacher/creator platform

## Development

Each application has its own README.md with specific setup instructions.

## Common Setup

All applications share common configuration and utilities from the `../shared` directory.
