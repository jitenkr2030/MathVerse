# MathVerse Creator Portal

Platform for teachers and content creators to manage educational content.

## Features

- Content creation tools
- Animation scene editor
- Revenue dashboard
- Analytics and insights
- Student engagement metrics

## Structure

- `src/`: Source code
- `components/`: Reusable components
- `pages/`: Application pages
- `services/`: API services

## Development

```bash
npm install
npm run dev
```
