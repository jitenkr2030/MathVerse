# Creator Portal Pages

This directory contains page components for the creator portal application.

## Pages

- **dashboard/**: Main dashboard and overview
- **content/**: Content creation and management
- **analytics/**: Analytics and insights
- **profile/**: User profile and settings
- **courses/**: Course management
- **revenue/**: Revenue and earnings

## Routing

Pages are connected using Next.js routing system.

## Authentication

Protected pages require authentication and role-based access.

## SEO

Pages include proper meta tags and SEO optimization.
