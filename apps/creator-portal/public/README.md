# Creator Portal Public Assets

This directory contains static assets for the creator portal.

## Contents

- **images/**: Static images and icons
- **fonts/**: Custom font files
- **favicon/**: Favicon files
- **manifest.json**: PWA manifest
- **robots.txt**: SEO robots file

## Usage

Assets are served from the public directory and accessible via URL.

## Optimization

Images are optimized for web performance.

## PWA

Includes Progressive Web App configuration for mobile experience.
