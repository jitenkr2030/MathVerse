# Creator Portal Source

This directory contains the source code for the MathVerse creator portal.

## Structure

- **components/**: Reusable React components
- **pages/**: Application pages and routes
- **services/**: API service functions
- **hooks/**: Custom React hooks
- **utils/**: Utility functions
- **styles/**: Styling and theme files

## Features

- Content creation tools
- Animation scene editor
- Revenue dashboard
- Analytics and insights
- Student engagement metrics
- Course management

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run start
```

## Testing

```bash
npm test
npm run test:coverage
```

## Deployment

Built for deployment on Vercel, Netlify, or similar platforms.
