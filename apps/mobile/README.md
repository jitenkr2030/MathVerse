# MathVerse Mobile Application

Flutter-based mobile app for the MathVerse platform.

## Features

- Cross-platform support (iOS/Android)
- Offline video downloads
- Progress synchronization
- Mobile-optimized UI
- Push notifications

## Structure

- `lib/`: Source code
  - `screens/`: Application screens
  - `widgets/`: Reusable widgets
  - `services/`: API services
  - `models/`: Data models

## Development

```bash
flutter pub get
flutter run
```
