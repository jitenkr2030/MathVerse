# MathVerse Web Application

Next.js-based web frontend for the MathVerse platform.

## Features

- Responsive design
- Course browsing and enrollment
- Video player integration
- Progress tracking
- User dashboard

## Structure

- `src/`: Source code
  - `components/`: Reusable components
  - `pages/`: Application pages
  - `services/`: API services
  - `store/`: State management
  - `styles/`: Styling
  - `utils/`: Utility functions

## Development

```bash
npm install
npm run dev
```
