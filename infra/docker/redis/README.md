# Redis Configuration

This directory contains Redis cache configuration for MathVerse.

## Files

- **redis.conf**: Redis configuration file
- **redis-cli.sh**: Command line interface script

## Usage

Redis is used for:
- Session storage
- API response caching
- Real-time data
- Job queues

## Configuration

- Memory management
- Persistence settings
- Security configuration
- Performance tuning

## Monitoring

Use Redis CLI tools for monitoring and debugging.
