# MathVerse Services Directory

This directory contains all microservices for the MathVerse platform.

## Services

- **animation-engine/**: Manim-based animation creation
- **video-renderer/**: Video processing and rendering
- **content-metadata/**: Content management service
- **recommendation-engine/**: ML-based recommendations

## Architecture

Each service is independently deployable and communicates via REST APIs.

See individual service README.md files for setup instructions.
