# MathVerse Animation Engine

Service for creating mathematical animations using Manim.

## Features

- Educational content creation
- Multi-level mathematics support
- Template-based animations
- Batch rendering capabilities

## Structure

- `manim_scenes/`: Animation content organized by educational level
- `templates/`: Reusable animation templates
- `renderer/`: Video rendering utilities
- `config/`: Configuration files
