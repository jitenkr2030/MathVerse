# Content Metadata Service Configuration

This directory contains configuration files for the content metadata service.

## Files

- **development.yaml**: Development environment config
- **production.yaml**: Production environment config
- **test.yaml**: Test environment config
- **logging.yaml**: Logging configuration

## Configuration

- Database connections
- API endpoints
- Cache settings
- Security settings
- Performance tuning

## Environment Variables

Use environment variables to override configuration values.

## Validation

Configuration is validated on service startup.
