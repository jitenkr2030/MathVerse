# MathVerse Video Renderer

Service for processing and rendering video content from mathematical animations.

## Features

- Asynchronous video processing
- Multiple quality options
- Thumbnail generation
- Progress tracking

## API Endpoints

- `POST /render`: Create new render job
- `GET /jobs/{id}`: Check job status
- `GET /video/{id}`: Download rendered video
- `GET /thumbnail/{id}`: Get video thumbnail
