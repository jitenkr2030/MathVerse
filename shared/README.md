# MathVerse Shared Directory

This directory contains shared code and configurations used across the MathVerse platform.

## Contents

- **schemas/**: Shared data models and schemas
- **utils/**: Common utility functions
- **constants/**: Shared constants and configurations

## Usage

Import from these directories to maintain consistency across all services and applications.
