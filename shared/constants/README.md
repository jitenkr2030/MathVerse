# MathVerse Constants

This directory contains shared constants used across the MathVerse platform.

## Contents

- **API Endpoints**: API URL constants
- **Error Codes**: Standardized error codes
- **Status Codes**: Application status constants
- **Config Values**: Default configuration values
- **Validation Rules**: Validation constraints
- **Color Schemes**: Brand color definitions

## Usage

Import constants from this directory to maintain consistency across the application.
