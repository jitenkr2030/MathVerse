# MathVerse Shared Schemas

This directory contains shared data models and schemas used across the MathVerse platform.

## Contents

- **User Schemas**: User-related data structures
- **Course Schemas**: Course and lesson models
- **Payment Schemas**: Payment and subscription models
- **API Schemas**: Request/response models
- **Database Schemas**: Database table definitions

## Usage

These schemas ensure consistency across all services and applications in the MathVerse ecosystem.
